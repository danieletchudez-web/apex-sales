Iniciar sesión con Google
Los datos se guardan automáticamente en Google Sheets y todos los miembros del equipo ven los mismos datos en tiempo real.
Si sos el primero en conectar, dejá vacío — se creará la Sheet automáticamente.
Luego compartí ese ID con tu equipo.
Cómo funciona para el equipo

1. El admin conecta primero → se crea la Sheet automáticamente
2. Va a Configuración → copia el Sheet ID que aparece
3. Comparte ese ID con cada miembro del equipo
4. Cada uno pega el ID arriba y conecta con su cuenta Google
5. ✅ Todos ven y editan los mismos datos en tiempo real
